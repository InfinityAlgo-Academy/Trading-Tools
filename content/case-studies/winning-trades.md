# Winning Trades

Analyze what works. Understand why trades win.

---

## Trade 1: Trend Pullback Buy

### Setup

```
Market: EUR/USD
Date: March 2024
Trend: Uptrend on daily
Structure: Higher highs, higher lows
```

### Entry

```
Entry: 1.0850
Trigger: Pullback to 50 EMA, bounce
Stop: 1.0800 (50 pips)
Target: 1.0950 (100 pips)
Result: +100 pips (WIN)
```

### Analysis

```
Why it worked:
- With trend
- Clear level (EMA support)
- Good risk/reward (2:1)
- Clear confirmation
```

### Lesson

```
Trade with trend, not against.
Use EMAs as dynamic support.
2:1 minimum on all trades.
```

## Trade 2: Range Breakout

### Setup

```
Market: GBP/JPY
Date: April 2024
Structure: Range for 3 days
Range: 188.00-188.50
```

### Entry

```
Entry: 188.55 (breakout)
Confirmation: Close above range
Stop: 187.90 (30 pips)
Target: 189.20 (70 pips)
Result: +70 pips (WIN)
```

### Analysis

```
Why it worked:
- Clear range boundaries
- Confirmed breakout
- Tight stop
- Good R:R
```

### Lesson

```
Trade broken ranges only.
Wait for close confirmation.
Use tight stops.
```

## Trade 3: RSI Divergence Reversal

### Setup

```
Market: AUD/USD
Date: February 2024
Trend: Range
Signal: Bearish divergence on RSI
Price: Lower low
RSI: Higher low
```

### Entry

```
Entry: 0.6550
Trigger: Trendline break, RSI confirmation
Stop: 0.6500 (50 pips)
Target: 0.6700 (150 pips)
Result: +150 pips (WIN)
```

### Analysis

```
Why it worked:
- Clear divergence
- Confirmation before entry
- Good risk/reward (3:1)
```

### Lesson

```
Divergence + confirmation = high probability.
Never trade divergence alone.
Minimum 2:1 R:R.
```
