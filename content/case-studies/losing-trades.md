# Losing Trades

Analyze what fails. Understand why trades lose.

---

## Trade 1: Fighting Trend

### Setup

```
Market: EUR/USD
Date: April 2024
Strategy: RSI reversal attempt
```

### Entry

```
Entry: 1.0950
Trigger: RSI at 30
Stop: 1.0900
Target: 1.1050
Result: -100 pips (LOSS)
```

### Why It Failed

```
Trend: Strong uptrend (not ranging)
RSI signal: Useless in strong trend
Entry: Against structure
```

### Lesson

```
Never reverse in strong trend.
RSI only works in ranging markets.
Check trend first.
```

---

## Trade 2: No Confirmation

### Setup

```
Market: GBP/JPY
Date: March 2024
Strategy: Breakout
```

### Entry

```
Entry: 188.00 (breakout anticipation)
Waited: Did NOT wait for confirmation
Result: -60 pips (LOSS)
```

### Why It Failed

```
Entry: Too early (anticipation)
Confirmation: None
Price: Pulled back into range
```

### Lesson

```
Wait for confirmation.
Never anticipate.
Enter on break, not before.
```

---

## Trade 3: Wrong Level

### Setup

```
Market: AUD/USD  
Date: February 2024
Strategy: Support bounce
```

### Entry

```
Entry: 0.6600
Support tested: Not real support (random level)
Stop: 0.6550
Result: -50 pips (LOSS)
```

### Why It Failed

```
Level: Not significant level
Swing high: Too far back
Identification: Wrong support
```

### Lesson

```
Use significant levels only.
Recent swing points.
Not random lines.
```

---

## Trade 4: Overstaying

### Setup

```
Market: EUR/USD
Date: January 2024
Strategy: Trend continuation
```

### Entry

```
Entry: 1.0800 (WIN +50 pips)
Action: Did NOT take profit
Target: Kept changing
Result: -20 pips (turned winner to loser)
```

### Why It Failed

```
Profit: Didn't take at target
Greed: "Let me make more"
Psychology: Won winner become loser
```

### Lesson

```
Take profit at target.
Trail stops, not targets.
Never let winner become loser.
```

---

## Trade 5: Size Increase

### Setup

```
Market: BTC/USD
Date: March 2024
After: Previous loss
```

### Entry

```
Previous: Lost $100
This: Size 2x to recover
Result: Lost $400 (LOSS)
```

### Why It Failed

```
Size: Increased after loss
Emotion: Revenge trading
Risk: Violated % risk rule
```

### Lesson

```
Never increase size after loss.
Always same % risk.
Walk away, don't chase.
```

---

# Key Takeaways

1. **Following trend** — Don't fight
2. **Wait confirmation** — Don't anticipate
3. **Use real levels** — Significant only
4. **Take profit** — Don't get greedy
5. **Fixed risk** — Don't increase size
