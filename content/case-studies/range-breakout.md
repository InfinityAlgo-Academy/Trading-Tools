# Range Breakout

Trading consolidations. Breakouts from ranges.

---

## Range Types

### Tight Range

- Narrow consolidation
- Energy building
- Breakout imminent

### Wide Range

- More volatility
- More time to break
- Clear edges

---

## Breakout Trading

### Long Setup

```
Range: Bounded
Break: Above resistance
Entry: On confirmation
Targets: Height of range
```

### Short Setup

```
Range: Bounded
Break: Below support
Entry: On confirmation
Targets: Height of range
```
