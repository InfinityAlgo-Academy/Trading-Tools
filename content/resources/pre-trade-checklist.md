## Checklist Summary

| Check | Description |
|-------|-------------|
| Setup | Clear setup name |
| Trigger | What triggers entry |
| Stop | Stop location (actual price) |
| Size | Position size calculated |
| Target | Target location |
| Risk | Dollar amount and % |
| Emotion | Calm or emotional? |
| Rules | Following system? |

---

## Quick Fill Guide

- **Setup**: Trend pullback, breakout, RSI reversal, etc.
- **Trigger**: Hammer candle, breakout close, crossover, etc.
- **Stop**: Below swing low, below support, etc.
- **Size**: Calculated from risk ÷ stop
- **Target**: 2:1 minimum, previous structure
- **Risk**: $ amount and account %
- **Emotion**: Calm ✓ / Anxious ✗ / FOMO ✗
- **Rules**: Following ✗ / Breaking ✗

---

## Quick Reminder

If ANY check fails → Don't trade.
