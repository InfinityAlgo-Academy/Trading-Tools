# Indicator Cheat Sheet

## Trend Indicators

| Indicator | Type | Use |
|-----------|------|-----|
| SMA (20, 50, 200) | Simple Moving Average | Identify trend direction |
| EMA (9, 21) | Exponential MA | Faster response |
| Ichimoku | Multi-line | Full market view |
| Parabolic SAR | Stop and Reverse | Trend reversal |

## Momentum Indicators

| Indicator | Use | Signal |
|------------|-----|--------|
| RSI (14) | Overbought/Oversold | <30 buy, >70 sell |
| MACD (12, 26, 9) | Trend/Momentum | Crossover |
| Stochastic | Momentum | Crossover <20 buy |
| ROC | Rate of Change | Extreme = reversal |

## Volatility Indicators

| Indicator | Use |
|-----------|-----|
| ATR (14) | Stop placement |
| Bollinger Bands | Range/overbought |
| Keltner Channel | Breakout |

## Volume Indicators

| Indicator | Use |
|-----------|-----|
| Volume | Confirm breakouts |
| OBV | Trend confirmation |
| VWAP | Fair value |

## Quick Reference

```
Uptrend: Price above 50 SMA, HH, HL
Downtrend: Price below 50 SMA, LH, LL
Range: Bounded between S/R

RSI < 30 = Oversold (buy)
RSI > 70 = Overbought (sell)
MACD cross up = Buy signal
MACD cross down = Sell signal
```

## Common Combinations

### Trend + Entry
```
50 SMA (direction)
RSI (entry timing)
Volume (confirmation)
```

### Range Trading
```
Bollinger Bands (bounds)
RSI (extremes)
Volume (confirmation)
```

### Momentum
```
RSI (overbought/oversold)
MACD (signal)
Volume (confirmation)
```

## Indicator Guide

| Goal | Indicator |
|------|-----------|
| Find trend | SMA, EMA |
| Time entry | RSI, Stochastic |
| Find reversal | RSI, MACD |
| Confirm breakout | Volume, ATR |
| Set stops | ATR, close |

---

*Keep it simple. 2-3 indicators max.*
