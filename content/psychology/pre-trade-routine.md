# Pre-Trade Routine

Your pre-trade routine prepares you for execution. Do it every time, consistently.

---

## Pre-Trade Routine

### Step 1: Check Environment

- Computer ready
- Data connected
- Noise minimized

---

### Step 2: Check Mental State

```
Am I emotional?
Am I focused?
Am I calm?
```

If not → Don't trade.

---

### Step 3: Check Market

- Is there a setup?
- Is my setup present?
- Is risk defined?

---

### Step 4: Check Rules

- Entry conditions met?
- Risk defined?
- Comfortable with risk?

---

### Step 5: Execute

- Enter with plan
- Wait for exit
- Follow rules

---

## Daily Routine Template

```
Morning:
☐ Review charts
☐ Check setup
☐ Define setup
☐ Check emotion

In-session:
☐ Check environment
☐ Execute only if calm
☐ Follow rules
☐ Record trade
```

---

# Key Takeaways

1. **Follow routine** — Every trade
2. **Check emotion** — Before executing
3. **Define setup** — Before entering
4. **Execute with discipline** — Rules first
