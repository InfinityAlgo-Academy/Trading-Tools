# Fear, FOMO, Greed

These three emotions cause more losses than any strategy failure.

---

## Fear

### Types

| Fear | Result |
|------|--------|
| Fear of losing | Avoid trades |
| Fear of being wrong | Exit early |
| Fear of missing out | Chasing price |

### Solution

```
Accept losses as cost
Focus on execution
Trust your process
```

---

## FOMO (Fear of Missing Out)

### Symptoms

- Entering late
- Chasing price
- "What if" thinking
- After-move entries

### Solution

```
Wait for your setup
Miss some moves
Stay patient
```

---

## Greed

### Symptoms

- Larger positions
- Holding too long
- Over-trading
- "One more trade"

### Solution

```
Take profits as planned
Follow size rules
Stick to limits
```

---

## Control Techniques

### 1. Pre-Trade Check

```
☐ Is this my setup?
☐ Am I calm?
☐ Is size correct?
☐ Is risk defined?
```

### 2. Pause When Emotional

```
Step away
Breathe
Reassess
Return only if calm
```

---

# Key Takeaways

1. **Recognize emotions** — Then control
2. **Use check** — Before every trade
3. **Accept variance** — Not every move
4. **Stick to rules** — No exceptions
