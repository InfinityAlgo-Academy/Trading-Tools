# How to Start Trading: Complete Beginner's Guide

This guide walks you through the entire process of becoming a trader - from having an idea to placing your first trade.

---

## Step 1: Education Before Anything

### What You Need to Know First

Before trading any real money, understand:

- [How markets work](/foundations/market-structure)
- [Risk management basics](/risk-management/position-sizing)
- [Your trading psychology](/psychology/trading-mindset)

```
Week 1-2: Read foundational materials
Week 3-4: Practice on demo account
Week 5+: Start live with tiny position
```

### Why Demo Trading Matters

- Practice without risk
- Test your platform
- Build confidence
- Learn execution

### Demo Trading Rules

```
✅ Trade exactly like live
✅ Use real position sizes (virtually)
✅ Record all trades in journal
✅ Treat like real money
```

## Step 6: Your First Strategy

### Simple Strategy to Start

**Trend Following:**

1. Daily chart shows uptrend (price above 50 SMA)
2. Price pulls back to 50 SMA
3. Bounce off 50 EMA with bullish candle
4. Enter on close above bounce candle
5. Stop below bounce low
6. Target: Previous high or 2:1

Full guide: [Trend Following Strategy](/strategies/trend-following)

---

## Step 7: Calculate Position Size

### Position Size Formula

```
Position Size = Risk Amount ÷ Stop Pips ÷ Pip Value

Example:
Account: $10,000
Risk: 2% = $200
Stop: 50 pips
Pip Value: $10/pip

Size: $200 ÷ 50 ÷ $10 = 0.4 lots
```

Full guide: [Position Sizing](/risk-management/position-sizing)

## Step 8: Place Your Trade

### Order Entry Checklist

```
☐ Confirmed trend direction
☐ Identified support/resistance
☐ Calculated position size
☐ Set stop loss
☐ Set take profit (minimum 1:1, ideal 2:1)
☐ Checked emotions (calm?)
☐ Placed order!
```

---

## Step 9: Manage Your Trade

### During Trade

```
DO:
✅ Watch price action
✅ Let stop work
✅ Trail stop if profitable

DON'T:
❌ Move stop
❌ Add to position
❌ Check every minute
```

---

## Step 10: After Trade

### Post-Trade Review

1. **Record in journal:**
   - Entry price, stop, target
   - Why you entered
   - How you felt

2. **Review:**
   - What worked?
   - What didn't?
   - What to adjust?

## Common Beginner Mistakes to Avoid

### 1. Starting Too Fast

**Mistake:** Real trading immediately.

**Solution:** Demo trade for 1-3 months first.

---

### 2. No Position Sizing

**Mistake:** Same size every trade.

**Solution:** Always calculate based on risk.

---

### 3. No Stop Loss

**Mistake:** Hope instead of stop.

**Solution:** ALWAYS use stops.

---

### 4. Overtrading

**Mistake:** Trading every day.

**Solution:** Wait for quality setups only.
