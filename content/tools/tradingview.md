## Chart Setup Guide

### Set Up Your First Chart

**Step 1: Select Symbol**
```
1. Click search box (top)
2. Type pair (e.g., "EURUSD")
3. Select from dropdown
```

**Step 2: Add Indicators**
```
1. Press Ctrl+I (or click fx icon)
2. Search for each indicator:
   - SMA (length 50)
   - SMA (length 200)  
   - RSI (length 14)
3. Adjust colors
4. Apply
```

**Step 3: Set Timeframe**
```
Quick access: Top bar
- D = Daily
- 4H = 4 Hour  
- 1H = 1 Hour
- 15 = 15 Minute
```
