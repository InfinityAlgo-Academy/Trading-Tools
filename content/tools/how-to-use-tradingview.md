# How to Use TradingView: Complete Guide

TradingView is the most popular free charting platform. Here's how to use it effectively.

---

## Getting Started

### 1. Create Account

1. Go to [tradingview.com](https://tradingview.com)
2. Create free account
3. Verify email

### 2. Navigate Interface

```
Top Bar:
- Symbol search
- Timeframes
- Indicators
- Full-screen

Left Sidebar:
- Chart tools
- Drawing tools

Right Sidebar:
- Watchlist
- Details

Bottom:
- Chart navigator
```

---

## Chart Setup

### Essential Indicators to Add

```
1. Press Ctrl+I or click indicator button
2. Search and add:

SMA (Simple Moving Average):
- Length: 50
- Color: Green

SMA (Simple Moving Average):
- Length: 200
- Color: Red

RSI:
- Length: 14
```

### Chart Timeframes

| TF | Use |
|---|-----|
| Monthly | Long-term direction |
| Weekly | Main trend |
| Daily | Swing trades |
| 4H | Entry timing |
| 1H | Intraday |

---

## Key Features

### 1. Symbol Search

Click symbol box → Type pair (e.g., EURUSD) → Select from dropdown

---

### 2. Drawing Tools

| Tool | Use |
|------|-----|
| Trend line | Draw trends |
| Horizontal line | S/R levels |
| Rectangle | Highlight areas |
| Text | Notes |
| Fibonacci | Retracements |

---

### 3. Indicators

Press Ctrl+I or:

1. Click indicator button
2. Search by name
3. Adjust settings
4. Apply

---

## Saving Your Setup

### Templates

1. Click chart
2. Save as template
3. Name it
4. Load on new charts

---

## Pro Tips

### Tip 1: Multiple Charts

Open same pair in different timeframes:
- Daily (direction)
- 4H (setup)  
- 15min (entry)

### Tip 2: Drawing Profiles

Save your S/R levels as named templates

### Tip 3: Alerts

Right-click → Set alert on any price level
