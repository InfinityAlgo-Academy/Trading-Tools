# Trading Systems

A trading system is a complete framework that combines strategy, risk management, psychology, and tools into one cohesive methodology. Without a system, you're gambling. With a system, you're trading.

---

# What Is a System?

A trading system is your complete business plan:

```
System = Strategy + Risk Management + Psychology + Tools

Without System:
- Random entries
- Emotional trades
- Undefined risk
- No edge

With System:
- Defined entries
- Controlled risk
- Psychology integrated
- Measurable edge
```

# Example System: Pullback Trend Following

## System Name

**"Trend Pullback System"**

## Market Conditions

- Timeframes: Daily (direction), 4H (entry)
- Markets: Forex majors, indices
- Market state: Trending (not ranging)

## Entry Rules

```
1. Identify uptrend: Price above 50 SMA, HH/HL
2. Price pulls back to 50 SMA or support
3. Wait for bullish candle/bounce
4. Enter on confirmation
```

## Risk Rules

```
- Stop: Below recent swing low
- Risk: Maximum 2%
- Target: New high or 2:1
```

## Timeframe Rules

```
- Check daily for trend direction
- Check 4H for entry timing
- Ignore lower timeframes
```

## Psychology Rules

```
- If emotional: Don't trade
- If revenge: Walk away
- After loss: Stop for day
```

## Execution Rules

```
- No pending orders
- Enter on close only
- No partial entries
```

## Journal Metrics

```
- Win rate: Track
- Average R:R: Track
- Consecutive losses: Track
- Emotion: Track
```
