# Strategy Tools

Tools used within strategies. Define what you need before trading.

---

## Tool Categories

### Trend Indicators

- SMA/EMA
- Ichimoku
- Parabolic SAR

### Momentum Indicators

- RSI
- MACD
- Stochastic

### Volatility Tools

- ATR
- Bollinger Bands

### Level Tools

- Pivot Points
- Support/Resistance
- Trend Lines

---

## How to Choose Tools

### Match to Strategy

| Strategy | Primary Tools |
|----------|---------------|
| Trend following | MA, MACD |
| Mean reversion | RSI, Bollinger |
| Breakout | Volume, pivot |
| Price action | Levels, candles |

---

# Key Takeaways

1. **Limit to 2-3 tools** — Max
2. **Each tool = one purpose** — Don't duplicate
3. **Master before adding** — Learn each
4. **Adjust to conditions** — Different markets
