# Exercises

Daily and weekly practice exercises. Build consistency through repetition.

---

## Daily Exercises

### Morning Exercise (10 min)

```
Before market:
☐ Review daily charts
☐ Mark key levels
☐ Identify setup(s)
☐ Define entry triggers
☐ Set alerts
```

### In-Session Exercise (Ongoing)

```
During market:
☐ Execute only on setups
☐ Check emotions
☐ Follow rules
☐ Record trades
```

### End-of-Day Exercise (15 min)

```
After market:
☐ Record all trades
☐ Note emotions
☐ Review P&L
☐ Plan tomorrow
```

---

## Weekly Exercises

### Weekly Review (Friday, 30 min)

```
At week's end:
☐ Review all trades
☐ Calculate win rate
☐ Calculate P&L
☐ Note emotional patterns
☐ Identify mistakes
☐ Adjust approach
☐ Plan next week
```

## Monthly Exercise

### Monthly Review (End of month)

```
Monthly:
☐ Review month's trades
☐ Calculate performance
☐ Adjust strategy if needed
☐ Evaluate mental state
☐ Set next month's goals
```

## Practice Drills

### Drill 1: Setup Identification

```
Task: Find 3 setups on charts
Time: 10 minutes
Done: Mark them
```

### Drill 2: Entry Execution

```
Task: Execute on setup trigger
Time: When triggered
Done: Enter only on signal
```

### Drill 3: Stop Management

```
Task: Move stops only
Time: As needed
Done: Trail only, never widen
```

### Drill 4: Emotion Check

```
Task: Check emotion before trade
Time: Before every entry
Done: Only trade if calm
```
