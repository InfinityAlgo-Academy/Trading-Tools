# Trade Setups

Master one setup at a time. Don't spread yourself thin — master the setup, then expand.

---

## Setup Mastery Process

### Process

```
1. Choose ONE setup
2. Practice ONLY that setup
3. Master it (50+ trades)
4. Then add another
```

---

## Popular Setups to Practice

### 1. Trend Pullback

```
Setup: Price pulls back to MA
Entry: On bounce
Stop: Below swing low
Target: Previous high
```

### 2. Range Break

```
Setup: Price consolidates
Entry: Breakout confirmation
Stop: Below lower range
Target: Range height
```

### 3. Support Bounce

```
Setup: Price tests support
Entry: On bounce
Stop: Below support
Target: Previous high
```

### 4. RSI Reversal

```
Setup: RSI at extreme
Entry: On confirmation
Stop: Beyond extreme
Target: Opposite extreme
```

### 5. Trendline Break

```
Setup: Trendline touch
Entry: On break confirmation
Stop: Beyond trendline
Target: Next structure
```

---

## Setup Practice Template

### Each Setup: Track This

| Field | What to Note |
|-------|-------------|
| Setup | Name |
| Entry trigger | What triggers entry |
| Stop location | Where stop |
| Target | Where target |
| Win rate | % wins |
| Avg win | Pips won |
| Avg loss | Pips lost |

---

## Mastery Checklist

```
☐ Can identify setup
☐ Can define entry trigger
☐ Can place stop
☐ Can set target
☐ Have 50+ trades on setup
☐ Win rate calculated
☐ Avg R:R calculated
☐ Consistent execution
```

---

# Key Takeaways

1. **One at a time** — Master one, add next
2. **50+ trades** — Minimum for mastery
3. **Track metrics** — Win rate, R:R
4. **Consistency** — Execute always
