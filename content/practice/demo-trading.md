# Demo Trading

Practice in risk-free environment before risking real money.

---

## Demo Trading Process

### Step 1: Choose Platform

| Platform | What's Good |
|----------|-------------|
| TradingView | Best charting |
| MetaTrader | Real execution |
| Broker demo | Real fills |

### Step 2: Set Rules

```
Same rules as live:
- Same setup
- Same position sizing
- Same risk
```

### Step 3: Track Metrics

```
Track:
- Win rate
- Profit factor
- Drawdown
```

### Step 4: Evaluate

```
After 50+ trades:
- Are results acceptable?
- Can you execute?
- Can you manage trades?
```

---

## Demo vs Live

| Aspect | Demo | Live |
|--------|------|------|
| Execution | Instant | Slippage |
| Emotions | None | Real |
| Pressure | None | Real |
| Results | Idealistic | Realistic |

---

## Demo Mistakes

### 1. No Tracking

**Problem:** Demo without metrics.

**Solution:** Track like real.

---

### 2. Different Rules

**Problem:** Sloppy demo rules.

**Solution:** Same rules as live.

---

### 3. No Emotion

**Problem:** Thinking demo = easy.

**Solution:** Psychology transfers.

---

# Key Takeaways

1. **Track everything** — Like live
2. **Same rules** — As live
3. **Minimum 50 trades** — For evaluation
4. **Then evaluate** — If profitable, go live
