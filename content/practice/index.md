# Practice Framework

## Daily Practice

```
Morning:
☐ Review charts
☐ Find setups
☐ Wait for signals
☐ Execute with rules

In-Session:
☐ Focus on execution
☐ Record everything
☐ Stay disciplined
☐ Accept outcomes

End of Day:
☐ Record all trades
☐ Note emotions
☐ Review what worked
☐ Plan tomorrow
```

## Weekly Review

```
Every Friday:
☐ Track win rate
☐ Note emotional patterns
☐ Identify mistakes
☐ Adjust approach
☐ Plan next week
```
