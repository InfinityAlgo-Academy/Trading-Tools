# Journaling

Your journal is your growth tool. Track everything, learn from everything.

---

## What to Track

### Trade Log

| Field | What to Record |
|-------|----------------|
| Date/Time | Entry date/time |
| Pair/Asset | What you traded |
| Setup | What setup triggered |
| Entry Price | Exact entry price |
| Exit Price | Exact exit price |
| Size | Position size |
| Direction | Long/Short |
| P&L | Profit or loss |
| P&L % | Account % change |

### Process Log

| Field | What to Record |
|-------|----------------|
| Pre-trade emotion | How you felt before |
| Post-trade emotion | How you felt after |
| Rules followed? | Yes/No |
| Why entered | Setup description |
| Why exited | Exit reason |

### Review Log

| Field | What to Record |
|-------|----------------|
| What worked | Wins |
| What didn't | Losses |
| What to adjust | Improvements |
| Pattern identified | Repeat issues |

# Key Takeaways

1. **Track everything** — Every trade
2. **Note emotions** — Know yourself
3. **Weekly review** — Identify patterns
4. **Monthly adjust** — Refine based on data
