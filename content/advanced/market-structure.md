## Wyckoff Method

### Wyckoff Phases

| Phase | What Happens |
|-------|---------------|
| Accumulation | Smart money buying |
| Uptrend | Distribution starts |
| Distribution | Smart money selling |
| Downtrend | Reaccumulation begins |

### Wyckoff Schematics

#### Spring

```
Price tests low
Bounces back
→ Bullish
```

#### Test

```
Price retests support
Holds
→ Bullish
```

#### UTAD (Upthrust After Distribution)

```
Price breaks up
Reverses
→ Bearish
```

## Industrial-Grade Concepts

### Delta

```
Delta = Volume at up ticks - Volume at down ticks
Positive = Buying pressure
Negative = Selling pressure
```

### Footprint Charts

```
     1.1085 ████████  500
     1.1080 ██████████ 800
     1.1075 ██        100
```

**Reading:**
- Large at bid = Buying pressure
- Large at ask = Selling pressure
